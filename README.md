# P2P Merchant Hunt Bot

## Setup

1. Replace `YOUR_BOT_TOKEN_HERE` in `p2p_bot.py` with your Telegram Bot Token
2. Deploy to Railway.app

## Deploy to Railway

1. Go to railway.app
2. New Project → Deploy from GitHub
3. Upload these files
4. Set environment variable: BOT_TOKEN=your_token

## Features
- 🚀 Start/Stop Bot
- ⚙️ Full Settings Menu
- 💳 Payment Methods Selection
- 🎯 Target Price Setting
- 📋 Statistics & Order History
- 🔑 API Key Management
- 🤖 Auto P2P Order Scanning
